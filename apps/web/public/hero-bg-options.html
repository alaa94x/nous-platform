02 — Recommended Combinations

Combination 01 — Best Overall
Ambient Glow + Dot Grid + Corner Orbital
Start with Option C (radial glow + dot grid - pure CSS, works everywhere). Then add Option D's corner orbital widget as a layer on top. The widget lives in the right half of the hero and doesn't compete with the headline. On mobile, the orbital collapses to static (just the logo mark, no rings). The glow and dot grid stay across all viewports. This is the safest high-impact path.
C - Radial glow C - Dot grid C - Terminal cursor on headline D - Corner orbital widget D - Magnetic pull on hover
Combination 02 — Most Unique
Circuit Lines + Data Pulse + Ghost Logo
Option F's PCB lines in the right half. Option A's ghost logo marks at ultra-low opacity, floating behind the circuit. Animated data-pulse dots. Result: a genuinely distinctive hero that explains what the company does without words. Higher implementation effort but visually sets Nous apart from every other agency in Qatar.
F - PCB SVG lines F - Animated pulse dots A - Ghost logos (opacity 5%) A - Mouse parallax
Combination 03 — Lowest Effort, High Return
Ghost Letter + Vertical Grid + Terminal Cursor
Pure CSS. Option E's giant "N" + vertical grid lines + the terminal cursor from Option C on the italic headline. Zero JS, zero performance cost, mobile perfect. Best ratio of work to visual improvement. Implement this first while deciding on the larger interactive elements.
E - Giant N E - Vertical grid C - Terminal cursor Existing grain layer

03 — Implementation Notes

Dot Grid (C)
Add directly to Hero.tsx as a position:absolute div inside the section. No component needed.
<div style={{
  position:'absolute', inset:0,
  backgroundImage: 'radial-gradient(
    rgba(10,92,71,.1) 1px, transparent 1px)',
  backgroundSize: '28px 28px',
  pointerEvents: 'none', zIndex: 0
}} />
Corner Orbital (D)
New client component: OrbitalWidget.tsx. Uses useMotionValue for the magnetic hover. position:absolute, right:56px, bottom:80px in the hero. Media query collapses rings on mobile.
import { motion, useMotionValue,
  useTransform } from 'motion/react'

const x = useMotionValue(0)
const y = useMotionValue(0)

onMouseMove: dx*0.25, dy*0.25
onMouseLeave: spring back to 0
Terminal Cursor (C)
Add a <span> after the italic headline word. Pure CSS blink. Uses existing blink keyframe already in globals.css.
<span style={{
  display:'inline-block',
  width: 3, height: 32,
  background:'var(--accent)',
  marginLeft: 6,
  animation: 'blink 1s step-end infinite'
}} />
