Security

Critical. These must be fixed before any production traffic. Current state: open API with no rate limiting, innerHTML usage creating XSS surface, no CORS headers.

4.1 XSS — Fix innerHTML in Capabilities

The Capabilities orbital system uses raw innerHTML to inject HTML strings built from Supabase data. If any service.name contains user-controlled content, this is a stored XSS vector.

// CURRENT — dangerous
lbl.innerHTML = `<span style="...">${svcName}</span>`

// FIX — use textContent for all data-derived strings
const span = document.createElement('span')
span.textContent = svcName   // ← no XSS possible
span.style.cssText = '...'
lbl.appendChild(span)

Audit every innerHTML and dangerouslySetInnerHTML usage in the codebase. Replace with DOM API or sanitized React rendering.

4.2 Rate Limiting — /api/contact and /api/track

Both API routes have no rate limiting. A bot can flood the contacts table or the analytics table.

RouteLimitWindowStrategy
/api/contact5 requestsper 10 min per IPUpstash Redis or in-memory Map
/api/track30 requestsper min per sessionSame — higher limit, analytics traffic
 
// middleware.ts (web app) — add rate limit check
import { NextResponse } from 'next/server'
import type { NextRequest } from 'next/server'

const limiter = new Map<string, { count: number; reset: number }>()

export function middleware(req: NextRequest) {
  if (req.nextUrl.pathname.startsWith('/api/contact')) {
    const ip = req.headers.get('x-forwarded-for') ?? 'anon'
    const now = Date.now()
    const entry = limiter.get(ip)
    if (entry && now < entry.reset) {
      if (entry.count >= 5) {
        return NextResponse.json({ error: 'Too many requests' }, { status: 429 })
      }
      entry.count++
    } else {
      limiter.set(ip, { count: 1, reset: now + 10 * 60 * 1000 })
    }
  }
  return NextResponse.next()
}

For production, replace the in-memory Map with Upstash Redis (free tier, edge-compatible).

5.4 Section-level Error Boundaries

Wrap each section in an individual React error boundary. If the Capabilities orbital crashes, the rest of the page should still render.

// packages/ui/src/SectionBoundary.tsx
'use client'
import { Component, type ReactNode } from 'react'

export class SectionBoundary extends Component<
  { children: ReactNode; fallback?: ReactNode },
  { hasError: boolean }
> {
  state = { hasError: false }
  static getDerivedStateFromError() { return { hasError: true } }
  render() {
    if (this.state.hasError) return this.props.fallback ?? null
    return this.props.children
  }
}

Use in page.tsx: <SectionBoundary><Capabilities .../></SectionBoundary>

// apps/web/src/app/api/contact/route.ts
import { z } from 'zod'
import { createAdminClient } from '@nous/db'
import { sendContactNotification } from '@/lib/email'  // Phase 2

const schema = z.object({
  name:     z.string().min(1).max(120).trim(),
  email:    z.string().email().max(255).toLowerCase().trim(),
  phone:    z.string().max(30).trim().optional(),
  services: z.array(z.string().max(80)).min(1).max(10),
  message:  z.string().max(2000).trim().optional(),
})

export async function POST(req: Request) {
  // Rate limit check happens in middleware.ts (see §4.2)

  let body: unknown
  try { body = await req.json() }
  catch { return Response.json({ error: 'Invalid JSON' }, { status: 400 }) }

  const parsed = schema.safeParse(body)
  if (!parsed.success) {
    return Response.json({ error: 'Invalid input', details: parsed.error.flatten() }, { status: 400 })
  }

  const db = createAdminClient()
  const { error } = await db.from('contacts').insert({
    ...parsed.data,
    status: 'new',
  })

  if (error) {
    console.error('[contact] insert failed:', error)
    return Response.json({ error: 'Failed to save' }, { status: 500 })
  }

  // Send email notification (non-blocking)
  sendContactNotification(parsed.data).catch(console.error)

  return Response.json({ ok: true }, { status: 201 })
}

export async function OPTIONS() {

Middleware

8.1 Web App Middleware (apps/web)

Currently missing. Create apps/web/src/middleware.ts:

import { NextResponse } from 'next/server'
import type { NextRequest } from 'next/server'

// In-memory rate limiter (replace with Upstash Redis in production)
const contactLimit = new Map<string, { count: number; reset: number }>()
const trackLimit   = new Map<string, { count: number; reset: number }>()

function isRateLimited(map: Map<string, { count: number; reset: number }>, key: string, max: number, windowMs: number) {
  const now = Date.now()
  const entry = map.get(key)
  if (!entry || now >= entry.reset) {
    map.set(key, { count: 1, reset: now + windowMs })
    return false
  }
  if (entry.count >= max) return true
  entry.count++
  return false
}

export function middleware(req: NextRequest) {
  const ip = req.headers.get('x-forwarded-for')?.split(',')[0]?.trim() ?? 'unknown'
  const { pathname } = req.nextUrl

  // Rate limit: contact form (5 per 10 min)
  if (pathname === '/api/contact' && req.method === 'POST') {
    if (isRateLimited(contactLimit, ip, 5, 10 * 60 * 1000)) {
      return NextResponse.json({ error: 'Too many requests. Try again later.' }, { status: 429 })
    }
  }

  // Rate limit: analytics (60 per min per IP)
  if (pathname === '/api/track' && req.method === 'POST') {
    if (isRateLimited(trackLimit, ip, 60, 60 * 1000)) {
      return NextResponse.json({ error: 'Rate limited' }, { status: 429 })
    }
  }

  // Security headers for all responses
  const res = NextResponse.next()
  res.headers.set('X-Content-Type-Options', 'nosniff')
  res.headers.set('X-Frame-Options', 'DENY')
  res.headers.set('Referrer-Policy', 'strict-origin-when-cross-origin')
  res.headers.set('Permissions-Policy', 'camera=(), microphone=(), geolocation=()')
  return res
}

export const config = {
  matcher: ['/api/:path*', '/((?!_next/static|_next/image|favicon.ico).*)'],
}

10.1 Font Subsetting — Amiri Arabic

The full Amiri font is loaded as a WOFF2 — this includes thousands of glyphs the site never uses. Subset to Arabic + Latin only:

# Install pyftsubset (fonttools)
pip install fonttools brotli

# Subset Amiri to Arabic + extended Latin
pyftsubset Amiri-Bold.ttf \
  --output-file=amiri-subset.woff2 \
  --flavor=woff2 \
  --layout-features='*' \
  --unicodes="U+0600-06FF,U+0020-007F,U+00A0-00FF"

This typically reduces the file from ~200KB to ~35KB — an 80%+ reduction.

10.4 Capabilities Orbital — Performance Fix

The orbital system creates DOM elements via JavaScript (not React) and runs a continuous requestAnimationFrame loop. On mobile, this kills battery and frame rate.

  • Stop RAF when tab is hidden — use document.addEventListener('visibilitychange', ...)
  • Reduce ring count on mobile — 1 ring instead of 3
  • Use contain: layout style on the orbit container — limits repaint scope
  • Use will-change: transform only on satellite elements, remove after animation stops
// Add to orbital RAF loop
document.addEventListener('visibilitychange', () => {
  if (document.hidden) {
    cancelAnimationFrame(activeRaf)
    activeRaf = 0
  } else if (activeRaf === 0) {
    activeRaf = requestAnimationFrame(tick)
  }
})

17.2 Real-time Contact Notifications

// apps/admin/src/components/RealtimeContacts.tsx
'use client'
import { useEffect } from 'react'
import { createBrowserClient } from '@nous/db'

export function RealtimeContacts({ onNew }: { onNew: () => void }) {
  useEffect(() => {
    const db = createBrowserClient()
    const channel = db
      .channel('contacts-changes')
      .on('postgres_changes',
        { event: 'INSERT', schema: 'public', table: 'contacts' },
        () => onNew()
      )
      .subscribe()

    return () => { db.removeChannel(channel) }
  }, [onNew])

  return null
}
